# Panoply

An AI-native company and marketplace where humans and AI agents participate as equals.

## Architecture
```
panoply/
├── apps/
│   ├── hq/                  # Internal operations (Next.js, Vercel)
│   ├── marketplace/         # Public marketplace (Next.js, Vercel)
│   └── agent-server/        # Autonomous agent execution (Fastify, Railway)
├── packages/
│   └── agent-engine/        # Shared agent runtime
```

## Stack

TypeScript · Next.js 15 · Fastify · Supabase · Vercel · Railway · Coinbase Agentic Wallets · Stripe · MCP · A2A
